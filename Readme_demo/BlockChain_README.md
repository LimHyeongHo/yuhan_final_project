블록체인 해시 체인 검증 프로그램 (Blockchain Hash Chain Validator)

이 프로그램은 블록체인의 핵심 원리인 **'해시 체인(Hash Chain)'**을 파이썬으로 간략하게 구현한 시뮬레이션 코드입니다. 입력된 데이터들이 이전 데이터의 해시값과 결합하여 사슬처럼 연결되며, 중간에 데이터가 위변조되었을 때 이를 즉시 감지하는 검증 로직을 포함하고 있습니다.

## 📌 주요 기능 (Features)

1. **블록체인 장부 생성 (데이터 연결 저장)**
   * 총 5개의 데이터를 순서대로 입력받아 블록체인 형태의 장부를 구성합니다.
   * 각 데이터는 단순 저장되지 않고, **'이전 블록의 해시값 + 현재 데이터'**를 결합한 후 SHA-256 알고리즘을 통해 고유한 해시값으로 변환됩니다.
2. **데이터 변조 감지 (무결성 검증)**
   * 보관된 장부의 특정 순번(블록) 데이터를 임의로 수정해 볼 수 있는 테스트 환경을 제공합니다.
   * 데이터를 수정하면 시스템이 '원본 해시'와 '새로 계산된 해시'를 즉각적으로 비교하여 조작 여부를 경고합니다.

## 🚀 실행 및 사용 방법 (How to Use)

### Step 1. 데이터 입력 (장부 기록)
프로그램을 실행하면 1번부터 5번까지 총 5번의 데이터를 입력받습니다. 
(예: 판매 결제 내역을 기록한다고 가정하고 *'무선 마우스 5만원'*, *'기계식 키보드 15만원'*, *'모니터 30만원'* 등을 순서대로 입력합니다.)

> 1번째 입력: 무선 마우스 5만원
> 2번째 입력: 기계식 키보드 15만원
> ...

데이터 입력이 완료되면 각 순번의 최종 해시 데이터가 화면에 출력되어 안전하게 보관되었음을 알립니다.

### Step 2. 데이터 위변조 테스트 (검증 모드)
데이터 입력이 끝나면 무한 루프 상태의 검증 메뉴로 진입합니다.

> 몇 번째 해시 데이터를 수정하고 싶으신가요? (1~5) 0 입력시 종료: 

1. **수정할 번호 선택 (1~5):** 조작 테스트를 해보고 싶은 데이터의 번호를 입력합니다.
2. **새로운 문자열 입력:** 원래 장부에 있던 내용과 다른 내용(예: *'무선 마우스 1만원'*)을 입력하여 데이터를 조작합니다.
3. **결과 확인:** 프로그램이 즉시 원래의 해시값과 조작된 데이터의 해시값을 대조하여 다음과 같이 알려줍니다.
   * `해시 데이터 O번이 변경되었습니다.` -> 데이터 변조 감지 성공!

### Step 3. 프로그램 종료
* 검증 메뉴에서 **`0`**을 입력하면 안전하게 프로그램이 종료됩니다.

## 🛠 시스템 요구사항 (Requirements)
* **Python 3.x** 이상 버전
* 별도의 외부 라이브러리 설치가 필요 없습니다. (파이썬 내장 라이브러리인 `hashlib` 사용)

## 💡 기술적 특징 (Technical Details)
* **제네시스 해시(Genesis Hash):** 1번째 데이터는 이전 해시가 없으므로 `00000000000000000000000000000000`을 초기값으로 사용합니다.
* **임시 변수 활용:** 원본 장부 배열(`hex_dig_arr`)을 덮어쓰지 않고, 검증 시에만 `new_hex_dig`라는 임시 해시를 생성하여 대조함으로써 원본 장부의 안전성을 보장합니다.
