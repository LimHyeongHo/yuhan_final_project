# 📚 대학생 공동주문 마켓 (College Student Joint Order Market)

> **디지털 큐레이션 기반 프리미엄 중고 도서 공동구매 플랫폼**  
> 본 프로젝트는 고해상도 디자인 프로토타입을 기반으로 한 UI/UX 구현 모델입니다.

---

## 🎨 디자인 및 개발 프로세스
- **Stitch AI 활용**: Stitch AI를 통해 생성된 고성능 HTML/Tailwind 디자인 소스를 기반으로 제작되었습니다.
- **React 전환**: 정적 HTML 코드를 React 컴포넌트(JSX) 구조로 변환하고, 재사용 가능한 컴포넌트 단위로 최적화하여 구현했습니다.
- **프리미엄 UI**: Glassmorphism, Neon Theme 등 최신 웹 디자인 트렌드를 반영하여 대학생 타겟의 세련된 감성을 극대화했습니다.

## 🔑 로그인 및 네비게이션 테스트 (현재 구현 범위)
현재 이 프로젝트는 **UI 디자인 구성을 우선으로 작업**되었으며, 실제 백엔드 연동 전 각 사용자별 대시보드 진입을 확인하기 위한 **기초 테스트 기능**만 포함하고 있습니다.

### 1. 진입 방법
- 메인 홈페이지의 `Get Started`, `지금 시작하기`, `참여하기` 등의 버튼을 클릭하면 로그인 페이지(`/login`)로 이동합니다.

### 2. 계정별 대시보드 테스트 가이드
로그인 창에서 아래의 특정 정보(ID/PW)를 입력하면 각 역할(Role)에 맞는 홈페이지로 접속할 수 있습니다.

| 역할 (Role) | 아이디 (ID) | 비밀번호 (PW) | 진입 페이지 |
| :--- | :--- | :--- | :--- |
| **구매자 (Buyer)** | `admin1` | `admin1` | `/buyer` |
| **판매자 (Seller)** | `admin2` |  `admin2` | `/seller` |
| **관리자 (Admin)** | `admin3` |  `admin3` | `/admin` |

### 3. 기능 안내
- **로그인/로그아웃**: 각 페이지 간의 권한 구분과 이동 경로를 테스트하기 위해 간단한 로직이 구현되어 있습니다.
- **UI 중심 작업**: 현재는 실제 결제나 실시간 채팅 데이터 연동 등 비즈니스 로직은 구현되지 않았으며, 전체적인 **UI 디자인과 레이아웃 구성**에 집중되어 있습니다.

## 🛠 기술 스택
- **Library**: React 19
- **Routing**: React Router 7
- **Styling**: Tailwind CSS
- **Design Concepts**: Bento Grid, Glassmorphism, Responsive Design

## 🚀 시작하기

### 1. 의존성 설치
```bash
npm install
```

### 2. 로컬 서버 실행
```bash
npm start
```
- 접속 주소: `http://localhost:3000`
